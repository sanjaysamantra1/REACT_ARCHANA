=============

3. Next.JS

create react project using 'create-react-app'
=============================================
1. install NodeJS (https://nodejs.org/en/download)
2. check if nodejs is installed?
   cmd (node -v)
3. check if npm is installed? 
    NPM - Node Package Manager (Place to search packages/modules required for a react project) 
   cmd ( npm -v )
4. create a react project. go to a folder where project should be created
   npx create-react-app MyProject1
5. go to the project folder and start the project
   cd MyProject1
   npm start
